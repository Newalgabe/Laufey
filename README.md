# Laufey — Fan Tribute

A single-page fan tribute to [Laufey](https://laufeymusic.com), built with pure HTML5/CSS3/vanilla JS.

## Features

- **Music player** — stream track previews via Deezer, with direct Spotify and Apple Music links
- **Setlist** — filter by mood, search, queue tracks
- **Album showcase** — clickable album art, track details, lyrics modal
- **Tour dates**, press grid, newsletter signup
- **Cookie consent banner**, dark orchestral theme
- **Fully responsive** — desktop to mobile

## Structure

- `laufey.html` — entire app (styles + markup + JS)
- `404.html`, `privacy.html`, `terms.html` — supporting pages

No build tools, no frameworks. Open `laufey.html` in any browser.
